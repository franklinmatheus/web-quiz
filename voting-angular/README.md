## Voting Angular

1.  **Install the dependencies.**

    Just:

    ```shell
    # Go to the directory
    cd quiz-angular/

    # Install the dependencies
    yarn
    ```

2.  **Run the angular project!**

    Run `yarn start` and go to `http://localhost:4200`

3.  **Check out the stories!**

    Run `yarn storybook` and go to `http://localhost:6006`
